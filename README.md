# PDSNet-II-TEST
 The dataset comprises both 2D intensity images and 3D range images, encompassing diverse and intricate scenarios, including markings, bridge joints, and mixed distress. This dataset aims to evaluate the model's robustness. The test set does not contain a 6-fold rotation transformation. Download link: https://uofi.box.com/s/v17kuterdn71eld7if6skdfv16ndxqzw
